# Flutter Simple Game App

## Overview

This project is a basic Flutter application that demonstrates a simple interactive game. The app increases a score each time the user taps a button.

It is designed to showcase state management and user interaction in Flutter.

---

## Features

* Interactive button-based gameplay
* Real-time score update
* Simple and responsive UI
* Demonstrates state management using `setState`

---

## Technologies Used

* Flutter
* Dart
* Material UI components

---

## Project Structure

* `main.dart`

  * Contains the entire application logic
  * Includes:

    * `MyApp`: Root widget
    * `GamePage`: Main game screen

---

## How It Works

1. The app launches with the GamePage screen.
2. The initial score is set to 0.
3. When the user taps the "Tap Me" button:

   * The score increments by 1
   * The updated score is displayed instantly on the screen

---

## How to Run

1. Install Flutter on your system.
2. Create a new Flutter project or replace the content of `main.dart` with this code.
3. Run the following command:

   ```
   flutter run
   ```
4. Launch the app on an emulator or physical device.
